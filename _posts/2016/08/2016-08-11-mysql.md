### MySQL 笔记

#### 初始化数据库

MySQL 5.7 

```
> cd $DATA_DIR
> mysqld --initialize
```
这里会生成一个临时密码，需要修改这个密码。

> alter user 'root'@'localhost' identified by 'password';


MySQL 5.6

```
> mysql_install_db --datadir=$DATA_DIR
```
