### eprof可以植入到程序代码中 
过程记下： 
1. 启动eprof服务器 

```
eprof:start()  
```

2. 开始profiling 

```
eprof:profile([self()])  
```

或者 

```
eprof:start_profiling([self()])  
```

3.停止profing 

```
eprof:stop_profiling()  
```

4.现在就可以查看并分析结果了（不要停止eporf） 

```
eporf:analyse()  
```

B、通过eprof:profile启动待测试程序 

```
1. eprof:start()  
2. eprof:profile([self()], Mod, Fun, Args) %启动待测试程序  
3. eprof:stop_profiling()  
4. eprof:analyse() 
```
